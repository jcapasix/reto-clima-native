# Reto Clima - Aplicación React Native

Aplicación móvil desarrollada con React Native que permite buscar el clima actual de cualquier ciudad utilizando la API de OpenWeatherMap.

## Características

- Búsqueda de clima por ciudad
- Visualización de temperatura actual
- Información de humedad
- Descripción del clima
- Manejo de errores
- Pruebas unitarias

## Requisitos Previos

- Node.js (versión 14 o superior)
- npm o yarn
- Una cuenta en OpenWeatherMap para obtener una API key gratuita

## Instalación

1. Clona o descarga este repositorio
2. Instala las dependencias:

```bash
npm install
```

## Ejecución

### Desarrollo

```bash
npm start
```

### Ejecutar en plataformas específicas

```bash
# iOS
npm run ios

# Android
npm run android

# Web
npm run web
```

## Pruebas

Para ejecutar las pruebas unitarias:

```bash
npm run test:coverage
```

Para ejecutar las pruebas en modo watch:

```bash
npm run test:watch
```
